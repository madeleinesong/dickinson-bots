# Weil data

copyright in some jurisdictions — this corpus is for **personal experimentation**.

`raw/`

  original repo (essays, notebooks, *Gravity and Grace*, etc.).

To regenerate `processed/`:
```
python scripts/prepare_data.py weil
```
